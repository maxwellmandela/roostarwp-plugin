== Description ==

Roostar WP is a powerful livechat widget for WordPress that provides real-time communication with your website visitors. Powered by chatGPT, an advanced language model, Roostar WP offers intelligent and responsive chat capabilities, enhancing your customer support and engagement.

== Features ==

    Real-time Communication: Engage with your website visitors in real-time, providing instant support and assistance.
    Intelligent ChatGPT Integration: Utilizes the power of chatGPT, an advanced language model, to understand and respond to user queries effectively.
    Customizable Widget: Customize the appearance and behavior of the chat widget to match your website's branding and design.
    Easy Installation and Setup: Seamlessly integrate Roostar WP into your WordPress website with a simple installation and setup process.
    User-friendly Interface: The intuitive interface allows both website administrators and visitors to interact effortlessly.
    Responsive Design: The chat widget adapts to different screen sizes, ensuring a seamless user experience across devices.
    Analytics and Reporting: Gain insights into user interactions, track performance metrics, and optimize your customer support strategies.
    Plugin Updates and Support: Stay up to date with the latest features and receive timely support for any issues you may encounter.

== Installation ==

    Upload the roostar-wp folder to the /wp-content/plugins/ directory or install the plugin directly from the WordPress Plugin Directory.
    Activate the plugin through the 'Plugins' menu in WordPress.
    Go to the Roostar WP settings page and customize the widget appearance and behavior according to your preferences.
    Add the widget to your website by placing the shortcode [roostar_wp_widget] in the desired location.

== Frequently Asked Questions ==

= How do I customize the widget appearance? =

You can customize the widget appearance by navigating to the Roostar WP settings page and modifying the available options for color, style, position, and more.

= Can I use Roostar WP on multiple websites? =

Yes, Roostar WP can be used on multiple websites. Simply install and activate the plugin on each website individually.

= Is Roostar WP compatible with other live chat platforms? =

Roostar WP is designed to work as a standalone live chat solution. It is not directly compatible with other live chat platforms.
